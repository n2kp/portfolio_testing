Justo

Justo is a website where small businesses or sole traders who wanted apps or websites, can go online and create a project which developers can bid on to win the work. It allowed the business/trader to create or improve their online presence, whilst at the same time, provide freelance work to developers. The app was created using an AngularJS frontend and Ruby on Rails backend API. The app incorporated many different skills and techniques such as OAuth 2.0 and an internal messaging system between users. User authentication was again deployed, but in a slightly different manner as the site would have one of two different kinds of visitors, customers and developers. Views would vary depending upon what kind of visitor the user was. The app also provides timeouts and deadlines on bids using moment.js and the database selected was a PostgreSQL one. I was involved in creating the entire backend API for the app, which included the authentication, Oauth, internal messaging system, models, controllers and serializers and database design. I also worked on the frontend features such as the filtering logic, form validation and error messages. I was also heavily involved in the styling, for which we create our own custom CSS which we used in conjunction with a BootStrap 4 Grid system and UI BootStrap for all UI styling such as modals and start rating.
