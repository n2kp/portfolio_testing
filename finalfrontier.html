Final Frontier

Combining two of my passions, photography and astronomy, Final Frontier can be described as an Instagram for astrophotographers. The app was fully RESTful and built with an Express and Node.js backend, using session-type authentication and Instagram OAuth, and a JavaScript/jQuery frontend. It allows users to login and upload photographs they have taken of celestial bodies, indicating where and when they were taken, so as to help the community of users spot them in the sky depending on where they are in the world. The app also uses a NASA API to bring in daily photos and information as well as a DarkSky weather API to give the user forecast information about cloud coverage at their location for stargazing purposes. Styling the app was done using a CSS framework called Bulma.
